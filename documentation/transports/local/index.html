Local

Mutagen’s most basic transport provides support for synchronizing with local filesystem locations.

Usage

Local filesystem endpoints can be specified to Mutagen’s create command using their local path in either an absolute or relative format. If a relative path is specified, then it will be resolved relative to the working directory in which the create command is issued before being used to create the session.
